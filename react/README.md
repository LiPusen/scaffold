# REACT  基于浏览器项目脚手架

This project was bootstrapped with [Create React App](https://github.com/facebookincubator/create-react-app).

> 依赖安装

```javascript
npm install
```

> 运行

```javascript
npm start
```

> 打包发布

```javascript
npm run build
```

注: 该项目使用webpack进行打包,拥有完整的注释,可扩展性较强,持续维护更新中...  欢迎大牛一起维护并给出指导意见
